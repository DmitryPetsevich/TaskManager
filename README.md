## 🎬 Task Manager

Мини-приложение для управления проектами и задачами. Также реализована поддержка зависимостей между задачами:

- Задачи могут зависеть друг от друга
- Циклические зависимости предотвращаются
- Порядок выполнения вычисляется топологической сортировкой

## 🧱 Стек

- React + TypeScript
- TanStack Query (React Query) — управление серверным состоянием
- Zustand — управление клиентским состоянием
- RHF + Zod — формы и валидация
- Tailwind CSS — стилизация
- RTL + Vitest — тестирование
- json-server — fake backend

## 🚀 Функции

- Управление проектами (CRUD)
- Управление задачами (CRUD)
- Зависимости между задачами (DAG)
- Вычисление порядка выполнения задач (topological sort)
- Проверка циклических зависимостей
- Авторизация (fake backend)
